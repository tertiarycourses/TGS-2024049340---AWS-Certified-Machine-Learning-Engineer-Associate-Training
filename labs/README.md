# AWS Certified Machine Learning Engineer Associate Lab Guide

This folder contains practical labs for TGS-2024049340 - AWS Certified Machine Learning Engineer Associate Training.

## Lab Sequence

| Lab | File | Outcome |
| --- | --- | --- |
| 1 | [Account Safety, IAM, S3 ML Data Lake, and Data Formats](lab-01-account-iam-s3-ml-data-lake.md) | Secure ML workspace baseline with S3 data structure, encryption, and data-format notes. |
| 2 | [Data Ingestion, Transformation, Feature Engineering, and Data Quality](lab-02-data-ingestion-feature-engineering-quality.md) | Data preparation workflow with feature engineering, labeling, quality, and bias checks. |
| 3 | [Modeling Approach, SageMaker Algorithms, JumpStart, and AI Services](lab-03-modeling-approach-sagemaker-ai-services.md) | Model and service selection matrix for ML and AI business scenarios. |
| 4 | [Training, Hyperparameter Tuning, and Model Registry](lab-04-training-tuning-model-registry.md) | Training plan with tuning strategy, model versioning, and registry workflow. |
| 5 | [Model Evaluation, Performance Metrics, and Bias Analysis](lab-05-model-evaluation-performance-bias.md) | Evaluation metrics, bias/explainability notes, and performance analysis checklist. |
| 6 | [Deployment Infrastructure, Endpoints, Batch Transform, and Auto Scaling](lab-06-deployment-endpoints-inference-infrastructure.md) | Inference deployment comparison and endpoint operations checklist. |
| 7 | [MLOps Pipelines, CI/CD, and Workflow Orchestration](lab-07-mlops-pipelines-cicd-orchestration.md) | ML workflow orchestration design with approval, CI/CD, and rollback points. |
| 8 | [Monitoring, Security, Cost Optimization, and Exam Review](lab-08-monitoring-security-cost-exam-review.md) | Monitoring, security, cost optimization, and timed exam review plan. |
